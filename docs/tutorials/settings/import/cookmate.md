---
title: Importing From Cookmate
description: Import recipes from Cookmate by uploading .mcb export files. Easily transfer your recipes to RecipeSage.
---

You can import recipes exported by Cookmate (valid export files end in `.mcb`) by uploading them via the import page.

[https://recipesage.com/#/settings/import/cookmate](https://recipesage.com/#/settings/import/cookmate)
